COMMAND PROMPT

BY NOX

The command prompt, at least on Windows, is a vital tool for hacking. A hacker can use the command prompt for all he or she needs. The hacker can access the command prompt very simply. What the command prompt is gateway to the
operating system of the computer. With the command prompt, you have a nodescript access to the system. The thing is about the command prompt, it has no GUI or Graphical User Interface,
so it's harder to use that the standard GUI of the computer. But with a few simple commands, you can do more than the
GUI. In this section of NoxNet, I will show you how to use the command prompt for hacking.

STEP 1: ACCESSING COMMAND PROMPT RESTRICTED

Some administrators hide the command prompt icon so most of us cannot get to the prompt. The best way to access it afterward is to use a
batch file. To create a this batch file, go to notepad and type the code:

@echo off
start cmd.exe
exit

And save the file as cmd.bat and run the file later to get the command prompt. The command prompt should look like this:
C:\directory\directory\>
Or you'll see a prompt like this:
C:\Windows\System32\>
